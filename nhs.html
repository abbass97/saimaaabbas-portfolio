Applying Python to NHS Dataset

Overview:

The NHS was concerned about increasing patient demand and a high number of missed appointments. As part of a data analytics assignment, I was tasked to investigate how the NHS can make better decisions around resource planning and patient engagement. This analysis explores appointment data, service utilisation patterns, and public sentiment via Twitter.

Based on the gathered insights, I was able to make the following recommendations:

  • Focus on better utilisation of current capacity rather than immediate expansion.
  • Use Twitter to promote services and engage patients with hashtags like #healthcare and #mentalhealth.
  • Introduce SMS reminders and an improved online booking system to reduce no-shows.
  • Train staff in accurate data collection to improve data quality.

Approach:

For this project Python, APIs and webscrapping were used.

To answer the NHS's questions, I used Python with Pandas, Seaborn, and Matplotlib to clean, analyse, and visualise four datasets: actual_duration.csv, appointments_regional.csv, national_categories.xlsx, and tweets.csv. Key steps included:

Data cleaning and preprocessing

  • Aggregating trends over time using .groupby()
  • Creating visualisations to identify usage and capacity patterns
  • Extracting and analysing top hashtags from Twitter data
